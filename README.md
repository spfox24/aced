# **Aced**

An application designed with the avid disc golfer in mind. Log rounds played, leave condition reports for courses, and log Aces all in one location.

## Version 1.0
Current build includes:
- New Score Log
- Score edit/delete
- Condition report log
- Ace Log per user
- Google LogIn
